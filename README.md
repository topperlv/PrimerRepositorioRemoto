## Empezando

Bienvenido al mundo de Java en VS Code. Aquí tienes una guía para ayudarte a empezar a escribir código Java en Visual Studio Code.

## Estructura de carpetas

El espacio de trabajo contiene dos carpetas por defecto, donde:

- `src`: la carpeta donde se almacenarán los archivos fuente
- `lib`: la carpeta para mantener las dependencias

Mientras tanto, los archivos de salida compilados se generarán en la carpeta bin de forma predeterminada.

> Si desea personalizar la estructura de carpetas, abra .vscode/settings.json y actualice allí la configuración correspondiente.

## Gestión de dependencias

La vista `JAVA PROJECTS` (PROYECTOS JAVA) le permite administrar sus dependencias. Puede encontrar más detalles [aquí](https://github.com/microsoft/vscode-java-dependency#manage-dependencies).
